Project: 
    Sample Code.

Description:
    This is a sample banking system that is written in C#. It's a console based app that demonstrates C# 
    functionality and syntax. Using a TCP server, the main loop listens for incoming connections. Only after the
    client sends the correct message that is parsed by a regular expressions is the customer added to the bank.
    The user interface presents the user with different options. Clients can make transactions to deposit into their
    accounts or do a withdrawl, clients can also transfer money from one account to the other.
